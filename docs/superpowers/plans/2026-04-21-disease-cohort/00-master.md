# 专病管理 — Master Plan

> **拆分为 5 个子计划，按顺序执行，每个独立可测试。**

| # | 子计划 | 内容 | 产出 |
|---|--------|------|------|
| 01 | DDL + 种子数据 | 建表、索引、疾病模板种子 | 3 张表 + 16 条模板 |
| 02 | 后端 CRUD + 匹配引擎 | Entity/Repo/Service/Controller + 匹配引擎 | 全部后端 API |
| 03 | 前端列表 + 弹窗 | DiseaseList 卡片列表 + 创建编辑弹窗 + ConditionBuilder | 列表页可用 |
| 04 | 前端详情 + 统计 + 导出 | DiseaseDetail 患者列表 + 统计看板 + CSV 导出 | 详情页可用 |
| 05 | AI 推荐 + 模板集成 | AutoComplete 模板匹配 + AI 推荐接口 + 前端集成 | 智能推荐可用 |

执行顺序：01 → 02 → 03 → 04 → 05
